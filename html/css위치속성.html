디스플레이 속성

블록속성 : p태그는 기본적으로 마진값을 가지고있다

인라인속성 : width와 height값을 가질 수 없다.
div는 기본적으로 블록속성을 가지고 있음 display의 inline-block을 사용하면 width와 height를 가지는 inline처럼 쓸 수 있다.

visiblity 속성확인

기본속성은 visible

hidden으로 숨김

none으로 없앰

Opacity 속성확인

: 0에 가까울수록 투명

position의 속성확인

인라인블럭으로 정적위치 : position의 기본값
상대위치 : 원래 가진 정적 위치의 값에서 이동한 위치
절대위치 : 값이 고정된 top과 left에서는 위치가 고정
브라우저의 크기에 따라서 bottom과 right는 위치가 변형
고정위치 : 브라우저 화면 상에서 고정된 위치

z-iindex는 기본값으로0을 가지고 있고 값이 크면 앞으로 작으면 뒤로간다.
각 요소들은 작성해준 순서대로 쌓인다. 가장 마지막에 작성해준 요소가 가장 위에 출력
